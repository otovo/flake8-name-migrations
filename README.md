# flake8-name-migrations

A flake8 plugin to make sure you give your Django migrations human-readable names.

## Installation

```
pip install flake8-name-migrations
```

## Codes

| Code   | Description                                         |
|--------|-----------------------------------------------------|
| NMI001 | Rename migration to something human readable |
